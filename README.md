**CustomSeekbarDemo**

Demonstration for customised seekbar

This project is about for using Seekbar in a customised way.

Features:

1. Set textview in thumb view of seekbar.
2. Smooth dragging of seekbar.

##### Android SDK
   * minSdk 26
   * targetSdk 32
   * compileSdk 32
   * gradle 4.1.3
   * kotline 1.4.21
   * gradlew wrapper 6.5
